# Crypto Currency Price Analysis

## Overview
This project analyzes cryptocurrency price trends using historical data of Bitcoin, Ethereum, and Cardano.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Dataset
- Bitcoin historical data
- Ethereum historical data
- Cardano historical data

## Analysis Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Price Trend Visualization
- Comparative Analysis

## Results
Identified price patterns and trends across major cryptocurrencies using historical market data.
